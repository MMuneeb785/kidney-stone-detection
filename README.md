# 🫘 Kidney Stone Detection AI

An AI-powered web app that detects kidney conditions 
from CT/X-ray images using deep learning.

## 🔍 Classes Detected
- 🔴 Cyst
- 🟢 Normal
- 🟠 Stone
- 🔴 Tumor

## 🚀 How to Run

### 1. Install Libraries
pip install -r requirements.txt

### 2. Run the App
python app.py

### 3. Open Browser
http://localhost:5000

## 🛠️ Technologies Used
- Python
- Flask
- ONNX Runtime
- ResNet-18
- PIL / NumPy
